📐 Show the math 
load_w     = pv + grid_import − grid_export
setpoint   = setpoint for active mode (Default=50, Vacation=0, Wide=200, Off=skip)
target_w   = load_w + setpoint
target_pct = clamp(target_w / nominal_w × 100, 0, 100)

Override priority (highest wins):
  1. Voltage protection: if max(L1,L2,L3) > threshold → target_pct = 0
  2. Budget: if enabled and budget exhausted → 0; else → 100
  3. Mode = off                                       → target_pct = 100
  4. No PV detected (PV<50 W, current limit ≥99%)     → target_pct = 100

After overrides:
  target_w = (target_pct / 100) × nominal_w   (kept consistent)
